본 라이브러리는 블랙로보틱스의 아두이노 8S쉴드에서 사용하기 위한 라이브러리 입니다 
아두이노 우노R3 및 아두이노 메가2560과 호환됩니다 
최대 8개의 서보모터와 1개 센서, 1개의 블루투스 모듈을 장착 할 수 있으며 
블루투스 기능과 센서 기능은 추후 업데이트 예정입니다
해당 라이브러리는 블랙로보틱스에 의해 작성되었으며, 재배포는 가능하나 반드시 출처를 남겨주세요

This library is for use in Arduino 8S Shield of Black Robotics 
Compatible with Arduino Uno R3 and Arduino Mega 2560 
It can be equipped with up to eight servomotors, one sensor, and one Bluetooth module 
Bluetooth and sensor functions will be updated later
The library is written by Black Robotics and can be redistributed, but please make sure to leave your source
